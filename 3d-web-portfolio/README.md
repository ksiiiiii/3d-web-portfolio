3D Портфолио | Квалификационный экзамен

Интерактивное веб-портфолио с 3D-моделями, созданное в рамках квалификационного экзамена по профессии "Оператор электронно-вычислительных и вычислительных машин".

Описание проекта

Проект демонстрирует навыки:
- 3D-моделирования в Blender
- Веб-разработки (HTML, CSS, JavaScript)
- Интеграции 3D-контента в веб-интерфейс
- Деплоя проектов на GitHub Pages

Используемые технологии

- Blender — создание и подготовка 3D-моделей
- HTML5 / CSS3— структура и стилизация сайта
- JavaScript — динамическая загрузка моделей
- Model Viewer (Google) — интерактивный просмотр 3D в браузере
- GitHub Pages — хостинг и автоматическая публикация

Структура проекта
